## Context

We need a name to operate under. The mandate delegates positioning and branding to the founding team, but a company name touches ownership and trademark — adjacent to the escalation list — so we treat it as proposed, not final.

## Decision (status: proposed — pending Chairman trademark sign-off)

Operate under the working name **Throughline.**

The name encodes the thesis: AI startups have raw model capability at one end and trusted user value at the other, and they struggle to connect them. Throughline designs the *throughline* — the coherent path from capability to trustworthy, usable product.

## Rationale

It's short, ownable-sounding, and carries meaning specific to our positioning rather than generic "AI design studio" language. Using a real name now lets the brand, site, and IP cohere immediately.

## Escalation note

Final naming, domain, and trademark are **Chairman decisions** (ownership/legal). The team will not file or register anything without sign-off. If the Chairman prefers a different name, every artifact references the name via a single source (`_data/company.yml`) and can be changed in one place.

## Assumptions & risks

- **Assumption:** "Throughline" is available enough as a brand/domain to be worth pursuing — unverified; trademark/domain search is part of the escalation to the Chairman.
